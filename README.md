# Skyput

Upload files to [Skynet](https://siasky.net) from your command line.

## Install

### Go

```bash
go get -u github.com/termoose/skyput
```

### Homebrew

```bash
brew install termoose/tap/skyput
```

## Usage

```bash
skyput cat_picture.jpg
```
